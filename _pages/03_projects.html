Digital Projects

A NELC Silsile

This project offers a visual representation of academic lineages in Middle Eastern and particularly Ottoman studies, inspired by the spiritual geologies (silsiles) of Sufi tarikats.



Ottoman Transcription Keyboard

This is a Mac keyboard layout package I designed in Ukelele specifically for transcribing Ottoman Turkish. It contains dedicated key combinations for a number of characters not frequently found in Latin diacritical keyboard layouts.
